# Licensing

Agent Trail uses a mixed-license layout.

The root `LICENSE` is Apache-2.0 so GitHub displays the license for the spec and schema contract clearly.

## Apache-2.0

The following are licensed under Apache-2.0:

- `spec.md`
- `schema.json`
- `packages/schema/`
- `docs/PRD.md`, `CONTEXT.md`, and documentation under `docs/`, unless otherwise noted

## MIT

Implementation packages are licensed under MIT unless otherwise noted:

- `packages/core/`
- `packages/types/`
- `packages/adapters/`
- `packages/redact/`
- `packages/cli/`
- `apps/website/`
- future implementation packages such as `packages/mcp/` and `apps/desktop/`

Each published package should declare its own license in `package.json`.

## Package Metadata

- `@agent-trail/schema`: `Apache-2.0`
- Implementation packages: `MIT`
